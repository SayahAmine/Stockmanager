Stock Manager (C + SQLite)

A simple terminal-based stock management system written in C using the SQLite database.
The program allows users to manage products, including adding, editing, deleting, searching, and monitoring low stock items.

Features
•	Add new products
•	View all products
•	Edit product information
•	Delete products
•	Search products by name or category
•	Low stock alert system
•	Persistent storage using SQLite database

Technologies Used
•	C Programming Language
•	SQLite3 Database

Standard C libraries:
•	stdio.h
•	stdlib.h
•	string.h
